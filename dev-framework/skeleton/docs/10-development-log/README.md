# 10-development-log — 開發日誌

Daily / weekly 嘅開發日誌。同 phase `progress.md` 區分:progress 係 per-phase 嘅結構化記錄;呢層係跨 phase 嘅時間線敘事(可選)。

## 建議結構
```
10-development-log/
├── 01-daily/       ← {YYYY-MM-DD}.md 每日 log(可選)
└── 02-weekly/      ← {YYYY-Wnn}.md 每週匯總(可選)
```

## 慣例
- 呢層係 **optional**。若 phase `progress.md` 已足夠追蹤,唔一定要另開日誌。
- 適合:需要跨 phase 回顧、或者向 stakeholder 定期匯報嘅項目。

_(空層 — 按需入住)_
