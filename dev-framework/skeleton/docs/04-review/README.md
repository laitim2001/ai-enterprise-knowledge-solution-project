# 04-review — 審查記錄

Code review / security review / architecture review 嘅產出記錄。

## 建議住客
- Security review report(每次審查一份,標日期)
- 大型 code review 嘅 findings 匯總
- Pre-release review checklist 結果

## 慣例
- 一次審查一份文件,命名 `{type}-review-{YYYY-MM-DD}.md`。
- Findings 若需要 follow-up → 開對應 bug/change + 喺 BACKLOG 登記(R7)。
